# developer-tools
在学习中开发中所遇到能够提高效率的工具,总结如下(不断更新中):
## Markdown
博客文章常用语法,有利排版,README文件也正是这个语法写的.
## LaTex
文章编辑神器,能够很方便地编辑排版复杂的数学公式.
## Cmake
项目编译工具,语法参考: [https://www.hahack.com/codes/cmake/](https://www.hahack.com/codes/cmake/)
## Vim
代码编辑器,在命令行中编辑代码,拥有很多高效的快捷键和插件. 语法参考: [https://www.runoob.com/linux/linux-vim.html](https://www.runoob.com/linux/linux-vim.html)
